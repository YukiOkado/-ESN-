# -ESN-
リカレントニューラルネットワークの一種であるエコーステートネットワークを用いて、
カオス時系列(1次元データ)の予測を行うためのプログラムです。
エコーステートネットワークは、入力層、リザバー層、出力層から構成されており、
学習はリザバー層と出力層を繋ぐ重みのみで行うため、高速な学習が可能となる。
本プログラムでは、活性化関数はtanh、損失関数は平均二乗誤差、評価指数は平均絶対誤差とした。

echo.py は、リザバー層のニューロンが全結合であるのに対し、
WSmodel.py は、ニューロンがスモールワールドネットワーク、
BAmodel.py は、ニューロンがスケールフリーネットワーク
を構成しており、それぞれのパラメータを変化させることで、予測精度を調べることができる。
